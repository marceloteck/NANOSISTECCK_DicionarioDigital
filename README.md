## 🚀 Stack

- **Backend:** Laravel 10, PHP 8.1+
- **Frontend:** Vue 3 + Inertia.js
- **Build:** Vite + Sass + Tailwind + Bootstrap
- **Auth:** Laravel Breeze
- **SEO:** Meta centralizada, JSON-LD, sitemap.xml e robots.txt dinâmicos
